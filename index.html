Presentation Detail

Abstract

Deep learning allows computational models that are composed of multiple processing layers to learn representation of data with multiple levels of abstraction. These methods have dramatically improved the state-of-the-art in speech recognition, visual object recognition, object detection and many other domains such a drug discovery and genomics. Deep learning discovers intricate structure in large data sets by using the backpropagation algorithm to indicate how machine should change its internal parameters that are used to compute the representation in each layer from the representation in the previous layer. In this presentation I am going to talk about Deep Learning. First I will talk about the differences between Deep Learning and Machine Learning. Then we will talk about Supervised learning and Convolutional neural network and Backpropagation and finally we will talk about the future of deep learning.
